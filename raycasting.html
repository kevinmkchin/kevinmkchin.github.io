Raycasting Demonstration

This is a demonstration of a video game graphics rendering technique known as Raycasting (find out more about raycasting here: Raycasting Explained). Inspired by John Carmack's implementation of raycasting in Wolfenstein 3D.


The TileMap editor used to make the level in the demo became a standalone TileMap/Level editor which I use for other projects. Bambuu TileMap Editor


[W][A][S][D] to move.
