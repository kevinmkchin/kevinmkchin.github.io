Dream of Flight

A 2D platformer game inspired by EXOK’s Celeste. Built with C# on the MonoGame Framework.

[A] to move left, [D] to move right, [J] to jump, [K] to toggle flying.

Levels for this game are made using an old level editor I made: Bambuu TileMap Editor
