Quick write up on setting up a new Windows computer with development tools that I use.

# setup.ps1

# https://scoop.sh/

Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
irm get.scoop.sh | iex

scoop bucket add main
scoop bucket add extras

scoop install main/cmder-full       # cmder     https://github.com/cmderdev/cmder comes with git-for-windows
scoop install extras/sublime-text   # subl      https://www.sublimetext.com/
scoop install extras/fork           # fork      https://git-fork.com/
scoop install extras/renderdoc      # renderdocgui | renderdoccli   https://renderdoc.org/
scoop install hugo                  # hugo      https://gohugo.io/

scoop install extras/googlechrome   # chrome
scoop install extras/discord        # https://discord.com/
scoop install extras/spotify        # https://spotify.com/
scoop install extras/paint.net      # https://getpaint.net/
scoop install extras/obsidian       # https://obsidian.md/
scoop install extras/vlc            # https://www.videolan.org/

scoop install extras/color-cop      # http://colorcop.net/
scoop install extras/deskpins       # https://efotinis.neocities.org/deskpins/


# Cmder

Write-Host "Running init.bat for Cmder"
& ($env:CMDER_ROOT + "\vendor\init.bat")

## add visual studio developer tools to user_profile.cmd   https://learn.microsoft.com/en-us/visualstudio/ide/reference/command-prompt-powershell?view=vs-2022

Add-Content ($env:CMDER_ROOT + "\config\user_profile.cmd") "`ncall `"C:/Program Files/Microsoft Visual Studio/2022/Community/Common7/Tools/VsDevCmd.bat`" -arch=amd64 `n"

## Other plugins to PATH

Add-Content ($env:CMDER_ROOT + "\config\user_profile.cmd") "`nset PATH=C:\Notes;%PATH% `n"


# StartUp programs

## deskpins shortcut
Copy-Item "$Home\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Scoop Apps\DeskPins.lnk" -Destination "$Home\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup"

Run the above script with PowerShell: ./setup.ps1 or in cmd.exe: powershell setup.ps1

Scoop adds User Envrionment Variables - make sure they are right.

  1. Download latest Visual Studio Installer

    • Install Visual Studio 2022 and cherry pick the components I want:
    • Usually Desktop development with C++, Game development with Unity and Game development with C++. Cherry pick the optional components I want.
    • Desktop development with C++ component includes the MSVC compilers and headers, CMake, Windows 11 SDK, and other nice things like C++ profiling tools for Visual Studio.

  2. Configure Windows Terminal Settings

    • Add a new profile by duplicating Command Prompt profile
    • Add to the new profile’s Command line: /k %CMDER_ROOT%\vendor\init.bat
    • Set Icon path to %CMDER_ROOT%\icons\cmder.ico, and set profile and tab names
    • Choose my Cmder (cmd.exe) profile as Startup/Default profile
    • Disable Appearance/Always show tabs
    • Configure other Default/Appearance settings

Sublime Text

  1. Open Sublime Text and enter license.
  2. Install Package Control
  3. Install Package: Sync Settings https://packagecontrol.io/packages/Sync%20Settings
  4. Sync Settings: Edit User Settings -> enter gist_id and PAT
  5. Sync Settings: Download

Other notes

  • When using Git-for-Windows (comes with Cmder-full) for the first time and prompted to authenticate, use Git Credential Manager.
  • When opening fork from the command line, it may fail to locate the proper git instance. Open Fork’s File/Preferences/Git window and set custom Git Instance to where git.exe is located on my machine e.g. C:\Users\Kevin\scoop\apps\cmder-full\current\vendor\git-for-windows\bin\git.exe.
  • Maybe update Windows PowerShell to latest PowerShell on Microsoft Store.
  • Download https://strlen.com/procrastitracker/ and merge in old database.
